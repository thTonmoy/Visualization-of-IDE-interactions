Welcome

This project aims to visualize how developers interact with IDE using msr 2018 interaction dataset and other developer data from hacker rank survey.

Integrated Development Environments are used extensively in software development. Therefore, it provides immense insight into how developers work if we can visualize how they interact with various IDE features. Visualizing these interactions will give us a clear picture of how development activities typically go on. This information is helpful to evaluate whether the existing IDE environments give developers ample support and improve existing tools.
